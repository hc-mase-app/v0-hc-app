# PWA Installation Guide - HC App

## Apa itu PWA?

Progressive Web App (PWA) adalah aplikasi web yang bisa di-install seperti aplikasi native di smartphone Anda. PWA memberikan pengalaman yang lebih baik daripada web2apk karena:

- ✅ Semua fitur web bekerja sempurna (termasuk download PDF)
- ✅ Tidak ada error blob URL
- ✅ Install langsung dari browser, tidak perlu APK
- ✅ Auto update otomatis
- ✅ Offline support
- ✅ Ukuran lebih kecil
- ✅ Performance lebih cepat

## Cara Install PWA di Android

### Metode 1: Chrome Android (Recommended)

1. Buka website HC App di Chrome Android: `https://v0-hc-app.vercel.app`
2. Tunggu beberapa detik, akan muncul popup "Install HC App"
3. Klik tombol **"Install"**
4. Aplikasi akan muncul di home screen Anda

### Metode 2: Manual Install

1. Buka website di Chrome Android
2. Klik menu (3 titik vertikal) di pojok kanan atas
3. Pilih **"Add to Home screen"** atau **"Install app"**
4. Klik **"Install"**
5. Aplikasi muncul di home screen

### Metode 3: Dari Settings Chrome

1. Buka Chrome Settings
2. Pilih **"Notifications"**
3. Aktifkan notifikasi untuk website HC App
4. Kembali ke website, akan muncul prompt install

## Cara Install PWA di iOS (iPhone/iPad)

1. Buka website di Safari (harus Safari, bukan Chrome)
2. Klik tombol **Share** (kotak dengan panah ke atas)
3. Scroll ke bawah dan pilih **"Add to Home Screen"**
4. Klik **"Add"**
5. Aplikasi muncul di home screen

## Cara Uninstall PWA

### Android:
1. Long press icon aplikasi di home screen
2. Pilih **"Uninstall"** atau drag ke "Uninstall"

### iOS:
1. Long press icon aplikasi
2. Pilih **"Remove App"**
3. Pilih **"Delete App"**

## Keuntungan PWA vs web2apk

| Fitur | PWA | web2apk |
|-------|-----|---------|
| Download PDF | ✅ Bekerja | ❌ Error blob |
| Install | ✅ Langsung dari browser | ❌ Perlu download APK |
| Update | ✅ Otomatis | ❌ Manual download |
| Ukuran | ✅ ~1-2 MB | ❌ ~10-20 MB |
| Offline | ✅ Support | ❌ Terbatas |
| Performance | ✅ Cepat | ❌ Lambat |

## Troubleshooting

### Tidak muncul prompt install?
- Pastikan menggunakan Chrome di Android atau Safari di iOS
- Pastikan website sudah HTTPS (Vercel sudah otomatis HTTPS)
- Clear cache browser dan coba lagi
- Pastikan tidak dalam mode incognito/private

### Aplikasi tidak muncul di home screen?
- Cek di app drawer/list semua aplikasi
- Restart device
- Install ulang

### Download PDF masih error?
- Pastikan menggunakan PWA yang sudah di-install, bukan browser biasa
- Update PWA dengan uninstall dan install ulang
- Clear cache aplikasi
